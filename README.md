# How to Install


```
NexaMerchant\CheckoutOneBuy\Providers\CheckoutOneBuyServiceProvider::class,
```
Add it to config/app.php $providers

# How to Install with Composer

```
composer require nexa-merchant/checkoutonebuy
```
